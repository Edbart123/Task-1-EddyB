# DecodeLabs Week 1 - Data Cleaning Project

## Project Overview

This project was completed as part of the DecodeLabs Data Science Internship Program.

The objective of this project was to clean and prepare raw data for analysis by identifying and correcting data quality issues.

## Objectives

- Remove duplicate records
- Handle missing values
- Standardize data formats
- Improve overall data quality
- Prepare the dataset for analysis

## Tools Used

- Microsoft Excel
- Data Cleaning Techniques
- Spreadsheet Functions

- Cleaned Dataset
- Documentation

## Skills Demonstrated

- Data Cleaning
- Data Preparation

## What I Learned

Through this project I learned how important clean data is for accurate analysis. I gained hands-on experience identifying inconsistencies, removing errors, and preparing datasets for future reporting and analytics.
